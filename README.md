# 🧮 Calculadora Web

Projeto de uma calculadora simples desenvolvida com **HTML, CSS e JavaScript**.

Este projeto foi criado para praticar conceitos básicos de desenvolvimento web, como manipulação do DOM, estilização com CSS e interação com botões usando JavaScript.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do projeto

```
calculadora/
│
├── index.html
├── style.css
├── script.js
└── favicon/
```

---

## ⚙️ Funcionalidades

* Inserção de números e operadores
* Operações matemáticas básicas:

  * Soma (+)
  * Subtração (-)
  * Multiplicação (*)
  * Divisão (/)
* Botão **C** para limpar a tela
* Botão **<** para apagar o último caractere
* Botão **=** para calcular o resultado
* Limite de caracteres para evitar quebra do layout

---

## 🎨 Interface

A interface possui:

* Fundo com **gradiente**
* Calculadora centralizada na tela
* Botões interativos com efeito **hover**
* Display alinhado à direita, semelhante a calculadoras reais

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Manipulação do DOM com JavaScript
* Uso da função `eval()` para resolver expressões matemáticas
* Estilização e layout com CSS
* Organização de arquivos em um projeto web
